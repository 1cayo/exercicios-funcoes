# 📚 Estudos em JavaScript - Funções Básicas

Este repositório contém exercícios práticos que estou desenvolvendo durante meus estudos de JavaScript básico, com foco em funções.

## 🚀 Conteúdos praticados

- Declaração de funções
- Parâmetros
- Return
- Operadores matemáticos
- Estruturas condicionais (if/else)
- Manipulação de strings
- Uso da propriedade `.length`

## 🧠 Exercícios desenvolvidos

✔ Soma de dois números  
✔ Verificação de número par ou ímpar  
✔ Cálculo de média de aluno  
✔ Comparação entre dois números  
✔ Contagem de caracteres de uma string  

## 🎯 Objetivo

Fortalecer a base lógica e consolidar os fundamentos da programação antes de avançar para:

- Arrays
- Loops
- Arrow functions
- Manipulação mais avançada de strings
